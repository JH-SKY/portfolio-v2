Issue 01 · 데이터 유실

로그 성공 1,282건,
실제 DB엔 300건

원인rollback이 이전 성공분까지 전량 되돌림
해결 UPSERT + 트랜잭션 범위 재설계
결과: 수집 1,282건 = 저장 1,282건. 유실 0건 달성
Issue 02 · 파일 파싱 실패

파일명은 .pdf
실제 형식은 HWP

원인확장자 기반 판단 + 전체 배치 롤백
해결 Magic Number + Savepoint (중첩 트랜잭션)
결과: 파싱 성공률 24% → 57%
Issue 03 · 배치 세션 끊김

200건 이후
DB 연결이 끊김

원인Supabase pgbouncer 5분 idle timeout
해결 100건/페이지 fresh session 구조
결과: 1,255건 Full Sync 안정화 (중복 제거 후 순수 공고 수)
