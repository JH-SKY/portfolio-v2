08 — BIZ-FUND-AI

기술 자랑이 아니라,
이 구조가 필요했습니다.

정책자금 질문은 표현이 다르고 의도도 복합적이어서,
단일 검색과 단일 LLM으로는 사용자 질문을 끝까지 설명하기 어려웠습니다.

WHY HYBRID RAG "창업 지원금"과 "초기창업패키지"를 같은 질문으로 찾아야 했습니다

단일 검색은 표현 차이에 약했습니다.
그래서 키워드 검색과 의미 검색을 결합해 관련 문서를 안정적으로 끌어올렸습니다.

ISSUE 01

Full Sync가
200~300건 이후 멈춤

문제

Supabase timeout + 장수 세션

해결

fresh session + pool 설정 재조정

파일명은 PDF인데
실제 형식이 다름

문제

파싱 실패 + 전체 롤백 위험

해결

Magic Number + Savepoint

성공 로그는 남는데

문제

rollback으로 이전 성공 데이터 유실

해결

UPSERT + 트랜잭션 범위 재설계
